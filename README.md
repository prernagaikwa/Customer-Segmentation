# 🚀 Customer Segmentation using Machine Learning

## 📌 Project Overview
This project is a web-based application that performs **customer segmentation** using Machine Learning.  
It helps businesses understand customer behavior based on **age, gender, income, and spending score**.

---

## 🧠 Technologies Used

- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Machine Learning: K-Means Clustering (Scikit-learn)
- Visualization: Matplotlib, Seaborn
- Deployment: Render
- Database: SQLite

---

## ⚙️ Features

- Upload customer data using CSV file
- Perform automatic customer segmentation
- Generate multiple graphs:
  - Customer Segmentation
  - Age Distribution
  - Gender Distribution
  - Spending vs Age
  - Income vs Spending
- Business-friendly explanations for each graph
- Simple and user-friendly interface

---

## 📊 How It Works

1. User uploads a CSV file containing customer data
2. Data is validated (Age, Gender, Income, Spending Score)
3. K-Means clustering is applied
4. Customers are grouped into clusters
5. Graphs are generated and displayed with explanations
